┌────────────────────────────────────────────────────────────────┐
│  🚀 QUICK START (UNTUK YANG TERBURU-BURU):                     │
└────────────────────────────────────────────────────────────────┘

Windows User:
1. Double-click: setup_project.bat
2. Buat database 'si_unh_db' di phpMyAdmin
3. Edit file .env
4. Run: python manage.py migrate
5. Run: python manage.py createsuperuser
6. Double-click: run_server.bat
7. Buka: http://127.0.0.1:8000/
